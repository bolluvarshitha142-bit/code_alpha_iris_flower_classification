# Iris Flower Classification

## Project Description
This project classifies Iris flowers into three species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

using the K-Nearest Neighbors (KNN) Machine Learning algorithm.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
Iris Dataset

## Model Used
K-Nearest Neighbors (KNN)
